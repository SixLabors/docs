    Struct Rgba1010102

    Packed vector type containing unsigned normalized values ranging from 0 to 1. The x, y and z components use 10 bits, and the w component uses 2 bits.

    Ranges from [0, 0, 0, 0] to [1, 1, 1, 1] in vector form.
