Struct Bgra5551

Packed pixel type containing unsigned normalized values ranging from 0 to 1. The x , y and z components use 5 bits, and the w component uses 1 bit.

Ranges from <0, 0, 0, 0> to <1, 1, 1, 1> in vector form.
